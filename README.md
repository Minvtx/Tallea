# Tallea Lore Package

## Purpose
This repository contains the foundation for a persistent startup-world centered on Tallea, a Buenos Aires-born fit-intelligence startup for online apparel commerce.

The package supports:
- an orchestrator agent
- persistent world state
- future cycle generation
- a public-facing startup website

## Core Runtime Rule
Do not generate cycles that only add atmosphere.
Every cycle should change state, shift pressure, create residue, or alter relationships.
