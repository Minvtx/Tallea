# Canon

Stable, read-only lore for the Tallea world. Never modified by the runtime.

## Files

- `world_constitution.md` — Foundational world rules.
- `world_day0.md` — The Day 0 narrative snapshot of the company and ecosystem.
- `world_rules.md` — Hard constraints (what must always be true).
- `startup_identity.md` — Tallea's brand, product, market position, and constraints.
- `core_characters.md` — Full bios for the five team members.
- `core_group_dynamics.md` — How the team functions and fractures together.
- `core_protagonist_requirements.md` — What every protagonist in this world must satisfy.
- `ecosystem_instruments.md` — The pressure-instruments available to the orchestrator.
- `world_instantiation_brief.md` — How to instantiate a new run from canon.
- `public_site_brief.md` — Allowed/forbidden public claims and tone rules.

## Usage
The orchestrator should not load these by default. Load only when a cycle needs depth that runtime docs don't cover.
